# 山兮的个人作品集网站

## 网站展示
<img width="1280" height="800" alt="desktop-home" src="https://github.com/user-attachments/assets/ad6447db-71e3-416d-82bc-c2c75f1dfbe7" />




这是一个可以展示个人经历、项目与技能的作品集网站，基于**React+Typescript**实现，支持与个人的github互通，也支持桌面端与移动端。



网站地址：**https://is-shanxi.github.io/Portfolio-site/**


## 本地部署方案
npm install      # 安装依赖

npm run dev      # 启动开发服务器，访问 http://localhost:5173

npm run build    # 构建生产版本到 dist/
